Habilidades Blandas

Proyectos

Pokedex

La aplicación de Pokedex en React permite a los usuarios buscar y ver información detallada sobre los Pokémon, como su nombre, tipo, estadísticas y habilidades, utilizando la PokéAPI. Los usuarios pueden filtrar los Pokémon por tipo. El diseño se ha realizado con CSS puro para un control total sobre el estilo y la estructura de la interfaz.

Rick and Morty

La aplicación de Rick and Morty en React permite explorar personajes, episodios y ubicaciones usando la Rick and Morty API. Los usuarios pueden buscar personajes y filtrarlos por especie o estado. El diseño se hace con CSS puro para un control total sobre la interfaz.

Galleta de la fortuna

La aplicación de Galleta de la Fortuna genera frases aleatorias al usuario, cambiando el título, la descripción y la imagen de fondo dentro del recuadro. El diseño se ha realizado con CSS puro para personalizar la interfaz.
